# Agentic & Orchestration Services (151-200)

This catalog details AI-native microservices focused on Autonomous Agents and Workflow Orchestration.

## 151. Task Planner Service
* **Description**: Takes a high-level goal and decomposes it into a structured dependency graph of executable sub-tasks.
* **Features**:
  * DAG (Directed Acyclic Graph) generation.
  * Resource estimation per task.
  * Parallelization identification.
* **Requirements**:
  * **Models**: GPT-4 or Claude 3 Opus.
  * **UI Flow**:
  1. Input: "Plan a marketing campaign for product X".
  2. Output: Gantt chart or Tree view of tasks.
  3. "Execute" button to send to agents.

## 152. Tool Selector Service
* **Description**: Matches a natural language task description to the most appropriate tool or API from a registry.
* **Features**:
  * Semantic matching of descriptions.
  * Parameter extraction suggestions.
  * Registry management.
* **Requirements**:
  * **Models**: Embedding-based retrieval + LLM ranking.
* **UI Flow**:
  1. Input: "Get the weather in Tokyo".
  2. Output: `Call WeatherAPI.get_forecast(city="Tokyo")`.

## 153. Short-term Memory Manager
* **Description**: Manages the context window for conversational agents, handling truncation and summarization.
* **Features**:
  * FIFO buffer.
  * Relevance-based retention.
  * Token counting and optimization.
* **Requirements**:
  * **Models**: Tokenizer utilities.
* **UI Flow**:
  1. API middleware.
  2. Logs show "Context pruned: 500 tokens removed".

## 154. Long-term Memory Manager
* **Description**: Stores and retrieves user facts and historical interactions using vector storage.
* **Features**:
  * Semantic search retrieval.
  * Time-weighted decay.
  * "Fact extraction" on write.
* **Requirements**:
  * **Infrastructure**: Vector DB (Pinecone/Milvus).
* **UI Flow**:
  1. User asks "What's my daughter's name?".
  2. Agent queries memory.
  3. Response: "Your daughter is Sarah."
